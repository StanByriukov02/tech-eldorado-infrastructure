## 🚀 СПОСОБ 1: ЧЕРЕЗ ПРОВОДНИК WINDOWS (САМЫЙ ПРОСТОЙ)

### Открыть папку проекта:
```
1. Открыть Проводник Windows (Win + E)
2. Перейти в адресную строку
3. Ввести: C:\Users\dammi\tech-eldorado-infrastructure
4. Нажать Enter
```

### Открыть инструкцию Supabase:
```
1. В папке tech-eldorado-infrastructure найти папку "setup"
2. Открыть папку "setup"
3. Найти файл "STEP_BY_STEP_SUPABASE.md"
4. Двойной клик → откроется в редакторе (VS Code, Notepad++, или браузер)
```

## 🚀 СПОСОБ 2: ЧЕРЕЗ VS CODE (РЕКОМЕНДУЕТСЯ!)

### Открыть весь проект:
```
1. Открыть VS Code
2. File → Open Folder (или Ctrl + K, Ctrl + O)
3. Выбрать: C:\Users\dammi\tech-eldorado-infrastructure
4. Нажать "Select Folder"
```

### Открыть инструкцию:
```
1. В VS Code в левой панели найти папку "setup"
2. Открыть папку "setup"
3. Кликнуть на "STEP_BY_STEP_SUPABASE.md"
4. Файл откроется в редакторе
```
