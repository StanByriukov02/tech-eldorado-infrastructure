## 📋 ШАГ 1: НАЙТИ SQL ФАЙЛ

### Способ 1: Через VS Code (РЕКОМЕНДУЕТСЯ)

```
1. Открыть VS Code
2. Открыть папку: C:\Users\dammi\tech-eldorado-infrastructure
3. В левой панели найти папку: drizzle
4. Открыть файл: 0000_thin_ulik.sql
5. Выделить ВЕСЬ текст (Ctrl + A)
6. Скопировать (Ctrl + C)
```

### Способ 2: Через Проводник

```
1. Открыть Проводник
2. Перейти: C:\Users\dammi\tech-eldorado-infrastructure\drizzle
3. Найти файл: 0000_thin_ulik.sql
4. Открыть в блокноте (двойной клик)
5. Выделить ВЕСЬ текст (Ctrl + A)
6. Скопировать (Ctrl + C)
```

### Способ 3: Через скрипт (САМЫЙ ПРОСТОЙ!)

```bash
cd C:\Users\dammi\tech-eldorado-infrastructure
node scripts/run-migration-manual.js
```

**Скрипт выведет весь SQL в консоль - просто скопируй оттуда!**

## 🚀 ШАГ 2: ОТКРЫТЬ SUPABASE SQL EDITOR

```
1. Зайти на: https://supabase.com/dashboard
2. Войти в свой аккаунт
3. Выбрать проект: Tech Eldorado
4. В левом меню найти: SQL Editor
5. Нажать на "SQL Editor"
```

═══════════════════════════════════════════════════════════════════════════════

## 📝 ШАГ 3: ВСТАВИТЬ SQL

```
1. В SQL Editor нажать в поле для ввода SQL
2. Вставить скопированный SQL (Ctrl + V)
3. Убедиться что весь SQL вставлен (прокрутить вниз)
```

## ▶️ ШАГ 4: ВЫПОЛНИТЬ SQL

```
1. Нажать кнопку "Run" (внизу справа)
   ИЛИ
   Нажать F5
   
2. Подождать несколько секунд
```

**Ожидаемый результат:**
```
Success. No rows returned
```

**Если видишь ошибку:**
- "relation already exists" → таблицы уже созданы, всё ОК!
- Другие ошибки → скопируй текст ошибки и сообщи мне
