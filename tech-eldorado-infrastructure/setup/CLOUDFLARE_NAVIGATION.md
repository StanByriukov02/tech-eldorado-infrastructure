# КАК НАЙТИ CLOUDFLARE PAGES - ДЕТАЛЬНАЯ НАВИГАЦИЯ

**ПРОБЛЕМА:** Не могу найти где настраивать Cloudflare Pages

═══════════════════════════════════════════════════════════════════════════════

## 🎯 СПОСОБ 1: ЧЕРЕЗ ВКЛАДКИ (САМЫЙ ПРОСТОЙ!)

**Ты сейчас на странице "Account home" → "Domains"**

```
1. Вверху страницы видишь вкладки:
   Domains | Developer Platform | Zero Trust
   
2. Нажать на вкладку "Developer Platform"
   
3. На странице Developer Platform:
   - Найти "Pages" в списке сервисов
   - Или в левом меню найти "Pages"
   - Нажать "Pages"
   
4. На странице Pages:
   - Нажать "Create a project" (большая кнопка)
```

═══════════════════════════════════════════════════════════════════════════════

## 🎯 СПОСОБ 2: ЧЕРЕЗ ЛЕВОЕ МЕНЮ

**В левом меню (где "Account home"):**

```
1. Найти секцию "BUILD"
2. В секции BUILD найти "Developer Platform"
3. Нажать на "Developer Platform"
4. В подменю найти "Pages"
5. Нажать "Pages"
```

═══════════════════════════════════════════════════════════════════════════════

## 🎯 СПОСОБ 3: ПРЯМАЯ ССЫЛКА

```
1. Открыть в браузере:
   https://dash.cloudflare.com/[account-id]/pages
   
2. Или:
   https://dash.cloudflare.com/pages
```

## 📋 ЧТО ДЕЛАТЬ ПОСЛЕ ТОГО КАК НАШЁЛ PAGES

### Если проект ещё НЕ создан:

```
1. На странице Pages нажать "Create a project"
2. Выбрать "Connect to Git"
3. Выбрать GitHub
4. Авторизовать (если нужно)
5. Выбрать репозиторий: tech-eldorado-infrastructure
6. Нажать "Begin setup"
```

### Если проект УЖЕ создан:

```
1. На странице Pages найти проект "tech-eldorado"
2. Нажать на него
3. Перейти в "Settings" (вкладка вверху)
4. Найти "Builds & deployments"
5. Настроить build settings
6. Найти "Environment variables"
7. Добавить переменные
```

═══════════════════════════════════════════════════════════════════════════════

## 🔍 ВИЗУАЛЬНЫЕ ПОДСКАЗКИ

**На странице "Account home":**
- Вверху: вкладки "Domains" | "Developer Platform" | "Zero Trust"
- Нажать "Developer Platform" → откроется страница с сервисами
- Найти "Pages" → нажать

**В левом меню:**
- Секция "BUILD" → "Developer Platform" → "Pages"
